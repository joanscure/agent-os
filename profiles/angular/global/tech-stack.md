# Tech Stack

- Angular 17+ with TypeScript (standalone components)
- Angular Signals for state management
- Angular Router for navigation
- RxJS for async/reactive patterns

## Styling

- Angular Material or Tailwind CSS

## Testing
- Jasmine + Karma (unit)
- Cypress (E2E)

## Other

- Angular CLI
