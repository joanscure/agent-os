# Tech Stack

## Frontend
- Angular 17+ with TypeScript (standalone components)
- Angular Signals for state management
- Angular Router + HTTP Client

## Backend
- NestJS with TypeScript
- Feature-based module architecture
- Class-validator + class-transformer for DTOs
- TypeORM or Prisma for data access

## Database
- PostgreSQL

## Testing
- Jest (NestJS unit + integration)
- Jasmine + Karma (Angular unit)
- Cypress (E2E)

## Other
- ESLint + Prettier
- Swagger/OpenAPI via @nestjs/swagger
