# Tech Stack

## Backend
- Node.js LTS with TypeScript
- Express or Fastify
- Layered architecture (routes → controllers → services → repository)

## Database
- PostgreSQL
- Prisma or Drizzle as ORM

## Testing
- Vitest or Jest (unit + integration)
- Supertest for HTTP endpoint testing

## Other
- ESLint + Prettier
- Zod for input validation
- Helmet + express-rate-limit for security
